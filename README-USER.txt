CLAUDE NOTIFY v3.0.2
====================
Nhận thông báo khi Claude Code xong task hoặc hỏi bạn.


CÁCH CÀI
--------
1. Chạy file: Claude Notify_3.0.2_x64-setup.exe
2. Nhấn Next -> Finish
3. App tự khởi động (tìm icon nhỏ góc dưới phải màn hình, cạnh đồng hồ)


LẦN ĐẦU DÙNG
-------------
1. Double-click icon tray -> cửa sổ Settings mở ra
2. Bật "Enable notifications"
3. Nhấn "Save Settings"

Xong! Giờ khi Claude Code xong task bạn sẽ nghe tiếng + thấy popup.

TRƯỜNG HỢP KHÔNG CÓ THÔNG BÁO
Lúc này hãy khởi động lại Claude Code (đóng/mở lại terminal hoặc VSCode)


GỠ CÀI
------
Settings > Apps > tìm "Claude Notify" > Uninstall
(Nên tắt "Enable notifications" -> Save trước khi gỡ)
